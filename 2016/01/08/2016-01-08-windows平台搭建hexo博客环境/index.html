windows平台搭建hexo博客环境

使用说明
  • Shaaron
  • blog
  • git
  • hexo
  • windows

本文记录初次使用博客环境搭建过程。平台：windows!

版本问题

git 2.0+
hexo 3.0+

安装Git

下载安装。（略）

安装Node.js

  1. Windows下安装Node.js比较简单，直接下载安装，next->next->…finish即可，安装完成后环境变量 已经自行配好。
    下载地址：http://nodejs.cn
  2. 检查是否安装成功。可使用命令：node -v，查看版本号。

安装npm

  1. 最新版本的Node.js已经集成了npm。
  2. 检查是否安装成功。可使用命令：npm -v,查看版本号。

安装hexo

  1. 在git bush中执行命令：npm install -g hexo
  2. 安装过程可能会出现问题，需要修改npm安装源，参考 http://www.cnblogs.com/trying/p/4064518.html

开始使用hexo（测试）

在你指定的某一文件夹下右键git bush,依次执行如下命令：

  • hexo init
  • npm install
  • hexo g
  • hexo s

    成功如下图所示，生成本地博客:http://localhost:4000

生成第一篇博客

  1. 设置SSH keys 详见 : http://jingyan.baidu.com/article/d8072ac47aca0fec95cefd2d.html
  2. 将SSH keys添加进github账户中（注意权限为读和写）
  3. 将coding上的代码clone到本地
  4. 加载node_modules，使用命令：npm install
  5. 在/source/_posts文件夹中新建博客。（编辑博客本人使用软件markdownPad）
  6. 编译博客使用命令：hexo g
  7. 部署到github上使用命令：hexo d
  8. 注意推送到coding上！
